# 핵심 기능
- 1~9의 서로 다른 수로 이루어진 3자리의 수 맞추는 게임
- 스트라이크, 볼, 낫싱 힌트 이용해서 컴퓨터 수 맞추기
- 컴퓨터는 1~9의 서로 다른 임의의 수 3개 선택 -> 사용자가 3개의 수 입력 => 컴퓨터가 입력한 숫자에 대한 결과 출력


# 프로그래밍 요구사항
- jdk 8버전
- indent(인덴트, 들여쓰기) depth를 2까지만 허용
- 3항 연산자 금지
- 함수가 한가지 일만 하도록 최대한 작게
- camp.nextstep.edu.missionutils에서 제공하는 Randoms, Console API를 활용해 구현


# 구현할 기능 목록
1. 나누기
- model, service, view, controller

## 게임 로직(Service)
- 1~9의 서로 다른 임의의 수 3개 만들기(컴퓨터)
- 사용자가 입력한 수와 컴퓨터 수끼리 비교 => 스트라이크, 볼, 낫싱 결과 계산

## 사용자 입력/출력(View)
- 사용자 수 입력받기(inputView)
- 결과 출력하기(outputView)

## 모델(Model)
- 숫자 객체(사용자 입력, 컴퓨터)
- 결과 저장 객체(스트라이크, 볼)
